PYTHON PROJECT

The image artist project was completed by my partner, Bradley, and myself. This project allowed to us to take one image and transform it into an edited version of that image. For programming and coding, we wrote in the language Python. We first selected an image which was an artistic drawing of Jose Altuve, and decided to first change the colored spots of the image. This was completed by using RGB variables which converted the original colored spots from orange to blue. Next, we decided to change the positioning of the leg in the bottom right corner to move left and down based off of the original image. We accomplished this by first choosing the size of the cropped piece, and decided to paste it by giving it coordinates to follow. Finally, we wrote the function "im.show()" in order to display the enhanced image. The last line of the coding that was written saves the new image into the folder "Altuve_Engstrom.JPG.
